<h1>CI c использованием github/workflows:</h1>

<p>При изменении файла index.html: <br>
 
 <b>0)</b>Запускается контейнер с nginx, в который монтируется новый index.html.<br>
 <b>1)</b>Проверяется, что nginx отдаёт этот файл (код ответа 200 и совпадение md5-сумм).<br>
 <b>2)</b>После проверки контейнер удаляется.<br></p>
В случае провала пп 1 и 2 отправляется письмо на почту
