CI c использование github/workflows:

При изменении файла index.html: 
 0)Запускается контейнер с nginx, в который монтируется новый index.html.
 1)Проверяется, что nginx отдаёт этот файл (код ответа 200 и совпадение md5-сумм).
 2)После проверки контейнер удаляется.
В случае провала пп 1 и 2 отправляется письмо на почту
